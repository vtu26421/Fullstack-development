# Event Ticket Booking System

A full-stack web application for booking college event tickets.

## Tech Stack
- **Frontend**: React 18 + TypeScript + Vite
- **Backend**: Spring Boot 3.x + Spring Data MongoDB
- **Database**: MongoDB
- **Email**: Resend API

## Project Structure

```
event-ticket-booking/
├── backend/          # Spring Boot application
├── frontend/         # React + TypeScript application
└── README.md
```
